org.wayround.aipsetup.buildingsite.isWdDirRestricted(path)[source]¶

This function is a routine to check supplied path is it suitable to be a building site.

List of forbidden path beginnings:

[
'/bin', '/boot' , '/daemons',
'/dev', '/etc', '/lib', '/proc',
'/sbin', '/sys',
'/usr'
]

This dirs ar directly forbidden, but subdirs are allowed:

['/opt', '/var', '/']
Parameters:path – path to directory
Return type:True if restricted. False if not restricted.
