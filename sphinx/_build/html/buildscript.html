org.wayround.aipsetup.buildscript.build_actions_selector(actions, action)[source]¶

Used by build_script_wrap() to build it’s valid return action list

Return type:None if error. tuple (actions, action), where action = None if action == 'complete'. If action == 'help', both values returned without changes. If action is one of actions, actions = [action]. If action is one of actions and action ends with + sign, actions = actions[(action position):]
org.wayround.aipsetup.buildscript.build_script_wrap(buildingsite, desired_actions, action, help_text)[source]¶

Used by building scripts for parsing action command

Parameters:
  • buildingsite – path to building site
  • desired_actions – list of possible actions
  • action – action selected by building script user
  • help_text – if action == ‘help’, help_text is text to show before list of available actions
Return type:

int if error. tuple (package_info, actions), where package_info is package info readen from building site package info file, actions - list of actions, needed to be run by building script
