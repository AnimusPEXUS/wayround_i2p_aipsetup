pack Module¶

Set of commands to perform package creation sequence

This module creates checksums, removes garbage, packs file and all other preparations and actions needed to build final package

destdir_verify_paths_correctness() Function¶

org.wayround.aipsetup.pack.destdir_verify_paths_correctness(buildingsite)[source]¶

Check for forbidden files in destdir

Files ['bin', 'sbin', 'lib', 'lib64', 'mnt'] are forbidden to be in DESTDIR root. This is a rule for aipsetup based distributions. Except is special cases, when this function is avoided.

destdir_set_modes() Function¶

org.wayround.aipsetup.pack.destdir_set_modes(buildingsite)[source]¶

Ensure all files (and dirs) in DESTDIR have 0o755 mode.

If You interested in defferent modes for files after package installation, read about post_install.py (script, which need to be placed in package and will be executed after package installation)
