source_name_parse() Function¶

org.wayround.aipsetup.name.source_name_parse(filename, modify_info_file=False, acceptable_version_number=None, mute=False)[source]¶

Parse source file name and do some more actions on success

If this function succeeded but not passed version check - return None.

If this function succided, passeed version check and `modify_info_file’ is True - update infofile in info directory.

If this function succided, return dict:

ret = {
    'name': None,
    'groups': {
        'name'              : None,
        'extension'         : None,

        'version'           : None,
        'version_list_dirty': None,
        'version_list'      : None,
        'version_dirty'     : None,

        'status'            : None,
        'status_list_dirty' : None,
        'status_dirty'      : None,
        'status_list'       : None,
        }
    }

Note

version numbers are always joined with `.’
